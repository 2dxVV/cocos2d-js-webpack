# cocos2d-js-webpack
Develop Cocos2d-js with Modern Web Way. You can use ES6+ syntax, npm libraries and Node Http Server. The Webpack4 and Babel7 are also worked together.

# Development Build
```
npm install
npm run dev

or

yarn install
yarn dev
```

# Production Build
```
npm install
npm run prod
npm run server

or

yarn install
yarn prod
yarn server
```

# Eslint Checker
```
npm install
npm run lint

or

yarn install
yarn lint
```
